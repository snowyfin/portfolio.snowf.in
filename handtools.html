Hand-Tools Project

This project was a project based around working with hand tools, and also getting to know how to be involved in the community.

I had a lot of fun creating this, even though using hand tools for hours was hell.

If I had a chance to do this project again, I do think that I would take it because it was an over all fun project to do, and I really enjoyed the outcome. I wanted to make something that would be a little bit unique, and I found this design. I did all of the measurements myself, and I think that it came together really well.
